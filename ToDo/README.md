# To-Do List App

A simple To-Do List application built using React and Vite.

## Features

- Add tasks
- Edit tasks
- Mark tasks as completed
- Delete tasks

## Technologies Used

- React
- Vite
- JavaScript
- CSS

## How to Run

```bash
npm install
npm run dev
```

Open the local server link shown in the terminal.
